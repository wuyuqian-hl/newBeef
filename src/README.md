# 项目所有准备工作

1. 安装 路由 router，less 与 less-loader, axios,框架 vant
2. 由于直接 npm 安装的 less-loader 版本过高，不支持 xxx 函数，所以需要卸载再安装一个低版本的 less-loader
3. 在 src 下新建一个 router 文件，然后将将路由放在这个文件下
4. 在主页下新建一个 vue.config.js，编写跨域的相关代码
